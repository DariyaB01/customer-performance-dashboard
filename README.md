Customer Performance Dashboard
A Power BI dashboard designed to analyze customer demographics, revenue, and behavioral insights.
It provides a clear view of age segments, gender-based earnings, customer profiles, and spending behavior.

Technologies Used
Power BI
DAX
Excel
SQL

Key Insights
The 31–40 age group contributes the highest revenue ($80M).
Female customers slightly outperform males in total earnings (50.3% vs. 49.7%).
71% of customers have children, with Australia leading this segment.

Dashboard Features
Customer segmentation (with/without children) for behavioral insights
Revenue analysis by age group
Gender-based earnings breakdown
Identification of top and loyal customers
